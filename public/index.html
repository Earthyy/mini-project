ขั้นตอนการ ดาวน์โหลด

เลือกติดตั้งตามระบบปฎิบัติการ(OS)ที่ตัวเองใช้ได้เลย

เลือกตำเเห่งที่จะดาวน์โหลดไฟล์

กดเปิดเพื่อดาวน์โหลดลงเครื่อง จากนั้นกด "กดเรียกใช้"

เลือกชนิดของภาษาที่ต้องการ ในที่นี้คือ C++ หรือจะเลือกตัวอื่นด้วยก็ได้ แล้วกด "Install"

หลังจากดาวน์โหลดเสร็จสิ้น ระบบจะให้รีสตาร์ท
เมื่อรีสตาร์ทเสร็จให้กดเข้าโปรมเเกรมใหม่อีกรอบเเล้วจึง loigin ผ่าน Microsoft

การใช้งานโปรแกรมเบื่องต้น

นำเมาท์ไปชี้ที่ add เเล้วเลื่อนซ้ายไปกด New Item

เลือก C++ files(.cpp) เเล้ว กด Add
