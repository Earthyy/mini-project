Loop For

การทำพีระมิด ด้วยเครื่องหมาย *(ดอกจันทร์)

Coding to C program

                    
    #include < stdio.h >

    int main() {

        int i;
        int j;
    
        for (i = 1; i <= 10; i++) {

            for (j = 1; j <= i; j++) {		

                printf(" *");

            }

            printf("\n");
        }

        return 0;
    }

โดย loop for นั้นเป็นการใช้คำสั่งทำการทำงานซ้ำๆตามเงื่อนไขที่เรากำหนด
หากเงื่อไขยังเป็นจริงอยู่ก็จะทำไปเรื่อยๆ เเต่หากเงื่อไขเป็นเท็จเเล้วจะหลุดออกจาก loop ทันที่ 

                    
            #include < stdio.h >

1) #include คือคำสั่งการใช้ function ของ library 

                    
            for (i = 1; i <= 10; i++) {
                ......
            }

2) การกำหนดเงื่อนไขให้ค่า i ทำงานวนลูปทั้งหมด 10 ครั้ง

โดยจะเพิ่มไปเรื่อยตามเงื่อนไข i++ 

                    
                    
            {
                for (j = 1; j <= i; j++) {		

                    printf(" *");

            }

            printf("\n");

3) เงื่อนไขที่ว่าด้วย การเเสดงผลของตัวเเปล j ในรูปเเบบของ *(ดอกจันทร์)
6) โดยทำงานวนลูปห้ามมากกว่าจำกัดของ i หรือ น้อยกว่าเท่ากับ 10 

                    
                return 0;

5) return 0; คำสั่งการรีเทริน์ค่ากลับไปยัง library เพื่อจบการทำงาน
