Loop For

การทำพีระมิด ด้วยเครื่องหมาย *(ดอกจันทร์)

Codding to C program

                    
    #include < stdio.h >

    int main() {

        int i;
        int j;
    
        for (i = 1; i <= 10; i++) {

            for (j = 1; j <= i; j++) {		

                printf(" *");

            }

            printf("\n");
        }

        return 0;
    }

โดย loop for นั้นเป็นการใช้คำสั่งทำการทำงานซ้ำๆตามเงื่อนไขที่เรากำหนด
หากเงื่อไขยังเป็นจริงอยู่ก็จะทำไปเรื่อยๆ เเต่หากเงื่อไขเป็นเท็จเเล้วจะหลุดออกจาก loop ทันที่ 

                    
            #include < stdio.h >

#include คือคำสั่งการใช้ function ของ library 

                    
            for (i = 1; i <= 10; i++) {
                ......
            }

การกำหนดเงื่อนไขให้ค่า i ทำงานวนลูปทั้งหมด 10 ครั้ง

โดยจะเพิ่มไปเรื่อยตามเงื่อนไข i++ 

                    
                    
            {
                for (j = 1; j <= i; j++) {		

                    printf(" *");

            }

            printf("\n");

เงื่อนไขที่ว่าด้วย การเเดงผลของตัวเเปล j ในรูปเเบบของ *(ดอกจันทร์) โดยทำงานวนลูปห้ามมากกว่าขีดจำกัดของ 

                    
                return 0;

return 0; คำสั่งการรีเทริน์ค่ากลับไปยัง library เพื่อจบการทำงาน
